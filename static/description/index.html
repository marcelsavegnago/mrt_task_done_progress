Task Done Progress Bar 100%

Set Task's Progress Bar to 100% when marked as done

This module adds the following feature to the "Project" app:

  • Module which sets the progress to 100% when a project task is set as Done in kanban state. If you don't want to manually add worked hours to a task when it's completed, but simply have it's progress set to 100%.

Set task as Done

Set task to done state in the kanban view

Task List

Task progress set to 100%, easy to notice in the tree view as the progress bar is full
